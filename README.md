## ğŸ› ï¸ Installation & Setup

### Prerequisites
- Node.js (v16 or higher)

### Installation Steps

1. **Clone the repository**
   ```bash

   cd "SR Elan"
   ```

2. **Install dependencies**
   ```bash

   yarn install
   ```
